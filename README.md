### @安装引入组件库
#### 安装
```shell
nodejs:16版本
npm i -g nrm
nrm use npm
npm i @wsk1113/vue-EPui-sk
```

#### 引入
项目入口文件main.js
```js
/* 将@steveouyang/super-ep组件库作为插件安装 */
import SuperEpPlugin from "@wsk1113/vue-EPui-sk"
app.use(SuperEpPlugin)
```
